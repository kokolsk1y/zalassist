## Что можно взять у Леманы и адаптировать

### 1. Готовые решения по сценариям (приоритет: ВЫСОКИЙ)
**У Леманы:** раздел `/proekty/` — дизайн-проекты комнат со списком товаров и ценами.
**Для нас:** Phase 4 "Готовые комплекты" — но можно усилить:
- Не просто статичные наборы, а **сценарные комплекты**: "Электрика для однушки", "Щиток на дачу", "Розетки для кухни"
- Каждый комплект = описание задачи + список товаров + общая стоимость
- **Параметрическая адаптация**: ввёл площадь → пересчитано количество кабеля и автоматов

### 2. Группировка по помещениям/задачам (приоритет: СРЕДНИЙ)
**У Леманы:** раздел `/rooms/` — товары по комнатам (кухня, ванная...).
**Для нас:** Аналог для электротехники — группировка по зонам:
- Кухня (розетки с заземлением, УЗО для влажных зон, кабель для плиты)
- Ванная (IP44 розетки, АВДТ, влагозащита)
- Щитовая (автоматы, шины, корпус щитка)
- Уличная (кабель для улицы, IP65 розетки)

### 3. Калькулятор (приоритет: СРЕДНИЙ)
**У Леманы:** калькулятор ламината (площадь → количество).
**Для нас:** Калькулятор электрики:
- Площадь квартиры → примерное кол-во кабеля, автоматов, розеток
- Мощность техники → сечение кабеля и номинал автомата
- Можно встроить в ИИ-диалог как инструмент

### 4. "Показать менеджеру" → полноценный итог (приоритет: ВЫСОКИЙ)
**У Леманы:** корзина с печатью/отправкой списка.
**Для нас:** Уже есть "копировать артикулы", но можно усилить:
- Красивая **итоговая карточка** на экране (артикулы + названия + количество)
- Менеджер просто смотрит на экран планшета — вместо бумажки с каракулями

### 5. Дисклеймер наличия + привязка к магазину (приоритет: НИЗКИЙ)
**У Леманы:** наличие в конкретном магазине на карточке товара.
**Для нас:** Пока нет данных о наличии (каталог статичный). На будущее:
- Интеграция с 1С → реальное наличие
- Пометка "есть на складе" / "под заказ"

## Что НЕ берём (нерелевантно для MVP)

| Фича Леманы | Почему пропускаем |
|---|---|
| 3D-планировщик | Слишком сложно, не наш формат |
| Программа лояльности | Это задача магазина, не приложения |
| Сканер штрихкодов | Требует камеру + базу штрихкодов, можно позже |
| NFC-ценники | Требует оборудование в магазине |
| B2B-портал | Не MVP, но идея на будущее |
| Маркетплейс | Не наш формат |
