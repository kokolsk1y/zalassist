# Конкурентный анализ: Лемана Про (бывш. Леруа Мерлен)

> Дата анализа: 2026-04-05

## О компании
- 112 магазинов, 11 дарксторов, 6 РЦ
- ~130 000–292 000 SKU
- Доля онлайн >11%
- 60% покупателей начинают онлайн → покупают офлайн
- 1 млн+ профи-клиентов (B2B = 30% выручки, цель 40% к 2027)

## Сайт (lemanapro.ru)

### Каталог
- 15+ категорий с подкатегориями
- Фильтры по параметрам, сортировка
- Карточка товара: цена, артикул, характеристики, отзывы, наличие в магазине
- Сравнение товаров
- Поиск по артикулу и штрихкоду

### Интерактивные инструменты
| Инструмент | Описание |
|---|---|
| 3D-планировщик кухонь | Онлайн-конструктор с визуализацией |
| Готовые решения (`/proekty/`) | Дизайн-проекты комнат со списком товаров |
| Калькулятор ламината | Расчёт количества по площади |
| Калькулятор стоимости проекта | Быстрый расчёт цены |
| Обустройство по комнатам (`/rooms/`) | Товары сгруппированные по помещениям |
| Онлайн-запись на услуги | Бронирование проектирования |

### Контент
- Советы "Как выбрать" — энциклопедия ремонта
- Пошаговые гайды по самостоятельному ремонту

## Мобильное приложение
- Сканер штрихкодов → цена, наличие, отзывы
- Карта магазина с навигацией по отделам
- Калькулятор ламината
- Электронная карта лояльности
- Push-уведомления по геолокации
- Списки покупок

## Услуги в магазинах
- Колеровка, резка дерева/стекла/столешниц, пошив штор, анализ воды
- Бесплатное проектирование кухонь/ванных (2D/3D)
- Сервис "Руки" — платформа заказа мастеров
- Онлайн-запись на услуги

## B2B (Ключ-карта ПРО)
- 4 уровня: Спец → Мастер → Профи → Эксперт
- Кэшбэк до 4%, возврат без чека 24 мес
- B2B-портал (b2b-lemanapro.ru) — электронные торги
- API для интеграций (developers.lemanapro.ru)
- Персональные менеджеры для крупных клиентов

## Программа лояльности (B2C)
- Ключ-карта "Для дома"
- Возврат до 12 мес (24 мес для собственных брендов)
- Бонусная программа

## AI и технологии (2024-2026)
- AI-помощник обрабатывает 2/3 клиентских запросов (поддержка, НЕ подбор товаров)
- Чат-боты в HR (138K взаимодействий)
- ML-прогнозирование спроса (GlowByte)
- Роботы Яндекса на складах (депалетайзер)
- **Умные NFC-ценники** (2026) — касание телефоном → страница товара
- Роботы-инвентаризаторы (в планах)
