# Discussion Log — Phase 2

**Date:** 2026-04-02

## Questions Raised and Answered

### Q: Что происходит при вводе текста на главной — поиск на месте или редирект?
**A:** Редирект на /search?q=текст
**Decision:** Главная = лендинг. Ввод и chips ведут на /search с query params.

### Q: Как показывать карточку товара — в списке, на отдельной странице, или bottom sheet?
**A:** Bottom sheet (модальное окно снизу) — привычный мобильный паттерн
**Decision:** Компактные карточки в списке + bottom sheet с полной информацией при клике. Пользователь спросил про скорость загрузки — всё мгновенно, данные уже в памяти.

### Q: Как работает "Показать менеджеру"?
**A:** Экран с крупным списком + кнопка "Скопировать"
**Decision:** Крупный текст для чтения консультантом с телефона клиента + clipboard API для копирования.
