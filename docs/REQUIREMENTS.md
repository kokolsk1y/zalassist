# Requirements — ZalAssist

## V1 — Must Ship

| ID | Requirement | Notes |
|----|-------------|-------|
| **UI-01** | Стартовый экран с 3 CTA: "Подобрать под задачу", "Найти товар", "Готовые комплекты" | Не пустой чат. B2B сразу попадает в поиск, B2C — в диалог |
| **UI-02** | Мобильный PWA-интерфейс в фирменных цветах ЭлектроЦентр | Синий #2b7de0, оранжевый #f15a2c, белый #ffffff. Tap target 48px+, крупный текст |
| **UI-03** | Установка через QR-код — без скачивания из магазина | QR ведёт на URL, PWA работает сразу в браузере |
| **UI-04** | Дата последнего обновления каталога в UI | Обязательно — снижает риск устаревших данных |
| **UI-05** | Дисклеймер "Наличие и цены уточняйте у консультанта" | Под каждой рекомендацией ИИ и в комплектах |
| **UI-06** | Кнопка "Показать менеджеру" — форматированный список товаров | Список артикулов для быстрого оформления консультантом |
| **CHAT-01** | Диалоговый подбор товаров через ИИ | Клиент описывает задачу → ИИ уточняет (макс 2 вопроса) → список товаров с артикулами |
| **CHAT-02** | Quick-reply chips после каждого ответа ИИ | Кнопки-подсказки для быстрого продолжения диалога (особенно для B2B) |
| **CHAT-03** | ИИ рекомендует ТОЛЬКО товары из каталога | System prompt запрещает генерацию артикулов вне списка |
| **CHAT-04** | Стриминг ответов ИИ (посимвольно) | SSE через Cloudflare Worker. Пользователь видит ответ сразу, не ждёт |
| **SEARCH-01** | Поиск по артикулу с нормализацией | Убрать дефисы/пробелы, регистронезависимый. "ВА47-29 1P 16А" → "ВА47-29-1P-16A" |
| **SEARCH-02** | Поиск по названию и описанию (нечёткий) | MiniSearch с fuzzy matching, работает офлайн |
| **SEARCH-03** | Словарь синонимов электротехники | Минимум 20-30 пар: дифавтомат/АВДТ, УЗО, ИЭК/IEK и т.д. |
| **SEARCH-04** | Zero-result с альтернативой | Никогда пустой экран: похожие товары + "Спросить ИИ" |
| **KIT-01** | Готовые комплекты с видимым составом | 5-10 комплектов на старте. Артикулы видны в карточке без клика |
| **KIT-02** | Параметрическая адаптация комплектов | Площадь → коэффициент → количество товаров. Таблица коэффициентов |
| **DATA-01** | Скрипт конвертации каталога из 1С в JSON | convert-catalog.js: CSV/Excel → catalog.json с нормализацией полей |
| **DATA-02** | Каталог ~1000 SKU как JSON в PWA | Загружается при первом визите, кэшируется Service Worker |
| **ARCH-01** | Cloudflare Worker как прокси к OpenRouter | API-ключ скрыт от клиента. Rate limit 20 req/IP/час |
| **ARCH-02** | Cloudflare AI Gateway для кэширования | Exact-match кэш: TTL 1ч для диалогов, 24ч для поиска |
| **ARCH-03** | Prefix caching system prompt | Статичный system prompt в начале промпта — экономия 70-80% input токенов |
| **ARCH-04** | Service Worker с офлайн-поддержкой | Cache-First для статики, Stale-While-Revalidate для каталога |
| **ARCH-05** | GitHub Pages + GitHub Actions деплой | Автодеплой при push в main |
| **SEC-01** | Защита от prompt injection | System prompt ограничивает роль; лимит 500 символов на запрос |
| **SEC-02** | Rate limiting по IP | 20 запросов/час на IP через Cloudflare Worker |
| **ANALYTICS-01** | Аналитика с первого дня (GA4 или Plausible) | Сканирования QR, сессии, запросы к ИИ, найденные товары |
| **ANALYTICS-02** | QR-код с UTM-метками | Для отслеживания источника трафика |

## V2 — Planned

| ID | Requirement | Notes |
|----|-------------|-------|
| V2-01 | Интеграция с 1С через API (реальное наличие) | Webhook из 1С → Cloudflare D1 |
| V2-02 | Подбор аналогов через ИИ | "Этого нет — что подойдёт вместо?" |
| V2-03 | Голосовой ввод (Web Speech API) | Риск: шум в зале |
| V2-04 | История сессий | Клиент может вернуться к предыдущему подбору |
| V2-05 | Миграция на Cloudflare Pages | Без лимитов bandwidth |
| V2-06 | Генерация комплектов ИИ на основе знаний консультантов | Расширение базы комплектов без ручной работы |
| V2-07 | Мультиязычность (казахский, узбекский) | Если есть спрос от B2B-клиентов |
| V2-08 | Планшет/киоск в зале | Когда QR-воронка доказана |

## Out of Scope

- Оформление заказа / оплата через приложение
- Личный кабинет / регистрация
- Админ-панель для управления каталогом
- Мультимагазинность
- Push-уведомления
- Рейтинги и отзывы на товары
- Интеграция с мессенджерами (Telegram, WhatsApp)
- Сравнение товаров
