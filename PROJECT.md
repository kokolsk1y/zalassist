# ZalAssist

## What It Is

ZalAssist — веб-приложение (PWA) для клиентов магазина «ЭлектроЦентр» (stv39.ru), доступное через QR-код на телефоне. ИИ-помощник помогает клиенту подготовиться до общения с консультантом: подобрать товары под задачу, найти по артикулу, получить готовый комплект. Не заменяет консультанта — экономит его время и сокращает ожидание.

## The Problem

Магазин «ЭлектроЦентр» работает по модели **шоурум + склад**: товары выставлены по 1 экземпляру, всё остальное на складе. Без консультанта купить нельзя — он формирует счёт и накладную через терминал (1С).

- 6-7 отделов, по 2 консультанта в каждом
- Когда консультант занят крупным B2B-заказчиком — другие клиенты ждут
- Клиенты часто не знают что именно им нужно → консультант тратит время на базовые вопросы
- 3 канала продаж (магазин, почта, сайт) — все проходят через 1С
- Предыдущий проект (SmartTV-экран в зале) не взлетел из-за вмешательства руководства

## Target Users

**B2B — оптовики и подрядчики (основной приоритет)**
- Электромонтажники, прорабы, закупщики
- Знают артикулы и характеристики
- Хотят: скорость и точность — быстро найти, проверить наличие, подобрать аналог

**B2C — физические лица**
- Частники на ремонте, домовладельцы
- Часто не знают что именно нужно
- Хотят: ориентацию и вовлечённость — помощь в выборе, понятные описания, готовые решения

## Success Criteria (v1)

- **15%+ клиентов** в зале сканируют QR и пользуются приложением
- Консультанты подтверждают что клиенты приходят **подготовленными** (знают артикулы)
- Клиент может **самостоятельно** найти товар по артикулу или описанию
- Клиент может получить **список товаров под задачу** через диалог с ИИ
- Руководство видит ценность → выделяет ресурсы на развитие

## Hard Constraints

- **Бюджет**: минимальный, ~$0 на инфраструктуру + $10 на OpenRouter (ИИ-запросы)
- **Команда**: 1 человек (разработчик) + Claude как кодер
- **Хостинг**: нет доступа к серверу магазина → отдельный хостинг (GitHub Pages + Cloudflare Workers)
- **Данные**: каталог из 1С доступен только через ручную выгрузку (CSV/JSON), не сразу
- **Формат**: QR-код → PWA на телефоне клиента (без закупки оборудования)
- **Срок теста**: 1-2 месяца

## Known Unknowns

- Точный формат выгрузки каталога из 1С (какие поля, сколько SKU)
- Сколько и каких готовых комплектов можно собрать с консультантов
- Как клиенты реагируют на QR — будут ли вообще сканировать
- Оптимальная модель ИИ для соотношения цена/качество на электротехническом домене
- Нужен ли офлайн-режим (Wi-Fi есть, но мобильный интернет тоже)

## Out of Scope (v1)

- Интеграция с 1С в реальном времени (наличие, цены онлайн)
- Оформление заказа / оплата через приложение
- Личный кабинет, история заказов
- Планшет/киоск в зале (только QR на телефон)
- Мультимагазинность (только ЭлектроЦентр)
- Админ-панель для управления каталогом (обновление вручную)

## Технические детали

### Архитектура

```
[Телефон клиента] → QR → [PWA на GitHub Pages] → [Cloudflare Workers] → [OpenRouter API]
                                                         ↓
                                                   [Каталог JSON]
```

### Стек

- **Фронтенд**: PWA — работает как сайт на телефоне клиента
- **Бэкенд**: Cloudflare Workers (бесплатно до 100k запросов/день)
- **ИИ**: OpenRouter API → экономная модель (Claude Haiku / GPT-4o-mini)
- **Хостинг фронта**: GitHub Pages (бесплатно)
- **Данные**: выгрузка каталога из 1С в JSON

### Роль ИИ

| Функция | ИИ | Без ИИ (fallback) |
|---------|-----|-------------------|
| Подбор под задачу | Диалог с LLM — понимает задачу, задаёт вопросы, собирает список | Невозможно качественно |
| Поиск по описанию | Нечёткий поиск, синонимы ("дифавтомат" = "АВДТ") | Только точный поиск |
| Подбор аналогов | Сравнение характеристик, предложение замены | Ручная таблица (ограниченно) |
| Адаптация комплектов | Подстройка под параметры (площадь, тип помещения) | Статичные наборы |

### Экономика ИИ

- Стоимость: ~$0.001-0.003 за диалог
- Бюджет $10 ≈ 3 000-10 000 диалогов
- API-ключ на бэкенде (Cloudflare Workers), клиент не видит

### Источники данных

- **1С** — учётная система (товары, наличие, цены)
- **Битрикс** — CMS сайта stv39.ru
- **MVP**: ручная выгрузка из 1С в JSON

### Информация о магазине

- **Название**: ЭлектроЦентр
- **Сайт**: https://stv39.ru/
- **Профиль**: электротехника, электрооборудование, строительные материалы
- **Формат**: шоурум + склад, 6-7 отделов
- **Системы**: 1С + Битрикс
- **Wi-Fi в зале**: есть
